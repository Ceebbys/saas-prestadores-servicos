{% icon "server" "w-4 h-4 text-slate-500" %} Variáveis — Evolution API (Render)

No Web Service da Evolution API no Render, adicione:

AUTHENTICATION_API_KEY=sua-chave-secreta
SERVER_URL=https://evolution.onrender.com
DATABASE_ENABLED=true
DATABASE_PROVIDER=postgresql
DATABASE_CONNECTION_URI=postgresql://...

Sem DATABASE_ENABLED, o WhatsApp desconecta ao reiniciar o serviço.

{% icon "book-open" "w-4 h-4 text-slate-500" %} Passos para Configurar

  1. 1 Deploy da Evolution API no Render com imagem atendai/evolution-api:v2.1.1
  2. 2 Adicionar EVOLUTION_API_URL e EVOLUTION_API_KEY nesta aplicação
  3. 3 Preencher o nome da instância e clicar Salvar
  4. 4 Escanear o QR Code exibido acima com o WhatsApp do celular
  5. 5 Configurar o webhook da instância com a URL ao lado (evento MESSAGES_UPSERT)
  6. 6 Pronto! O chatbot já responde automaticamente.
