{% icon "book-open" "w-4 h-4 text-slate-500" %} Guia de Configuração

  1. 1

    Deploy da Evolution API

    No Render.com, crie um novo Web Service com a imagem Docker:

    atendai/evolution-api:v2.1.1

    Adicione a variável de ambiente:

    AUTHENTICATION_API_KEY=sua-chave-secreta
  2. 2

    Configure esta aplicação

    No painel do Render desta aplicação, adicione:

    EVOLUTION_API_URL=https://sua-evolution.onrender.com
    EVOLUTION_API_KEY=sua-chave-secreta
  3. 3

    Crie a instância

    Preencha o "Nome da Instância" no formulário ao lado e clique em Salvar.

  4. 4

    Escaneie o QR Code

    Acesse o Evolution API Manager, abra sua instância e escaneie o QR Code com o WhatsApp do celular.

  5. 5

    Configure o Webhook

    No painel da Evolution API, aponte o webhook da instância para a URL acima com evento MESSAGES_UPSERT.

  6. 6

    Pronto!

    Envie uma mensagem para o número conectado e o chatbot responderá automaticamente.
